## 🚀 ¿Cómo usar este repositorio?

1. Cloná el repositorio:
```bash
   git clone https://github.com/Jorge-Ruiz-Troccoli/Data-Science-III.git
```
2. Navegá a la carpeta del módulo que corresponda.
3. Abrí el notebook `.ipynb` con Jupyter Lab, Jupyter Notebook o Google Colab.
4. ¡Practicá y experimentá con el código!
